  2. ICSI201 Объект хандлагат программчлал хичээлийн хувьд
    • Cэдэв: Лаборатори-9 Удамшил ба объект хоорондын харьцаа (Inheritance, Aggregation, Composition)
      GitHub: Төсөлийн Github холбоос
      C++ хэл дээр объект хандлагат программчлалын үндсэн ойлголтуудыг бодит жишээгээр хэрэгжүүлсэн.
      Class диаграмм боловсруулж, классуудын хоорондын inheritance, aggregation, composition харьцааг зөв загварчилсан.
      Employee, Person, Spouse, Child, Division, JobDescription зэрэг классуудыг тодорхойлж, объектуудын уялдаа холбоог кодоор хэрэгжүүлсэн.
      Файлаас өгөгдөл уншиж (ifstream), олон объект үүсгэн өгөгдөл удирдах логик боловсруулсан.
      Багаар ажиллаж, Git/GitHub ашиглан хамтын хөгжүүлэлт, merge conflict шийдвэрлэх туршлага хуримтлуулсан.
      Кодын бүтэц, логик холбоо, объектын амьдралын мөчлөгийг (object lifecycle) ойлгож хэрэгжүүлсэн.
    • Cэдэв: Лаборатори-10 Загвар класс (Template), давхар холбоост жагсаалт (Doubly Linked List)
      GitHub: Төсөлийн Github холбоос
      C++ хэл дээр template класс ашиглан төрөл бүрийн өгөгдөл хадгалах давхар холбоост жагсаалт (linked list)-ыг өөрийн гараар хэрэгжүүлсэн.
      Жагсаалтын үндсэн үйлдлүүд болох add, insert, get, delete, length функцуудыг загвар класст тодорхойлон хэрэгжүүлсэн.
      Node, linkedList<T> зэрэг классуудыг ашиглан өгөгдлийн бүтэц, заагч (pointer), санах ойн удирдлагыг бодитоор хэрэгжүүлсэн.
      Shape, TwoDShape, Square, Circle, Triangle зэрэг классуудыг ашиглан полиморфизм, inheritance, abstract class ойлголтуудыг хэрэглэсэн.
      Дүрсүүдийг талбай болон периметрийн утгаар эрэмбэлэх (sorting) логик боловсруулсан.
      Багаар ажиллаж, Git/GitHub ашиглан хамтын хөгжүүлэлт, merge conflict шийдвэрлэх туршлага олж авсан.
  3. ICSI202 Өгөгдлийн бүтэц хичээлийн хүрээнд
    • Сэдэв: Шугаман өгөгдлийн бүтэц (Array, Linked List, Stack, Queue)
      GitHub: Төслийн GitHub холбоос
      C/C++ хэл дээр массив, нэг болон давхар холбоост жагсаалт (singly/doubly linked list), стек (stack), дараалал (queue) зэрэг шугаман өгөгдлийн бүтцийг хэрэгжүүлсэн.
      Өгөгдөл нэмэх, устгах, хайх, урт тодорхойлох зэрэг үндсэн алгоритмуудыг pointer ашиглан бодитоор хэрэгжүүлсэн.
      Санах ойн динамик хуваарилалт (malloc/new, free/delete) ашиглаж, санах ойн алдаа (memory leak)-аас сэргийлэх дадал эзэмшсэн.
      Алгоритмын гүйцэтгэлийн хугацаа (time complexity) болон санах ойн ашиглалтын ялгааг харьцуулан шинжилсэн.
    • Сэдэв: Мод ба эрэмбэлэх алгоритмууд (Tree, Search & Sorting)
      Binary Tree, Binary Search Tree (BST) бүтэц дээр өгөгдөл оруулах, хайх, устгах алгоритмуудыг хэрэгжүүлсэн.
      Bubble Sort, Selection Sort, Insertion Sort, Merge Sort зэрэг эрэмбэлэх алгоритмуудын логикийг хэрэгжүүлж, гүйцэтгэлийг харьцуулсан.
      Алгоритмын онолын ойлголт (Big-O notation)-ыг практик кодтой уялдуулан ашигласан.
      Багаар ажиллаж, Git/GitHub ашиглан кодын хувилбарын хяналт (version control), хамтын хөгжүүлэлтийн туршлага хуримтлуулсан.
